# 🚀 快速启动指南

## 立即运行应用

### 1️⃣ 进入项目目录
```bash
cd my-onchainkit-app
```

### 2️⃣ 安装依赖
```bash
npm install
```

### 3️⃣ 启动开发服务器
```bash
npm run dev
```

### 4️⃣ 打开浏览器
访问 [http://localhost:3000](http://localhost:3000)

---

## 📝 下一步

1. **浏览排行榜**: 访问 `/rankings` 查看活动排行
2. **连接钱包**: 点击右上角钱包按钮连接 Web3 钱包
3. **查看文档**: 阅读 `DEVELOPMENT.md` 了解详细开发指南
4. **开始开发**: 编辑 `app/` 目录下的文件开始自定义

---

## 💡 提示

- 项目使用 **OnchainKit** 实现 Web3 功能
- 默认连接到 **Base** 链（可在 `rootProvider.tsx` 修改）
- 所有页面支持响应式设计（移动端友好）
- 使用 **CSS Modules** 进行样式隔离

---

## 🆘 遇到问题？

1. 确保 Node.js 版本 >= 18
2. 删除 `node_modules` 和 `package-lock.json` 重新安装
3. 检查端口 3000 是否被占用
4. 查看终端错误日志

---

**准备好了吗？运行 `npm run dev` 开始你的 Web3 之旅！** 🚀
